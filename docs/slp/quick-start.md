# Quick Start Guide

Get SexLab P+ up and running in just a few steps.

---

## Prerequisites

Before installing SexLab P+, ensure you have:

- **Skyrim Special Edition 1.6.1170** (recommended) or 1.5.97 (limited support)
- A mod manager (MO2 or Vortex)
- [SKSE](https://skse.silverlock.org/) for your Skyrim version
- [SkyUI](https://www.nexusmods.com/skyrimspecialedition/mods/12604) for MCM menus

---

## Installation Steps

<ol class="steps">
<li>
<strong>Install Base SexLab</strong><br>
Install the base SexLab Framework first as a precaution. SLP+ will overwrite everything except the logo.
</li>

<li>
<strong>Install Mfg Fix NG</strong><br>
Download and install <a href="https://www.nexusmods.com/skyrimspecialedition/mods/114095">Mfg Fix NG</a> and its requirements.
</li>

<li>
<strong>Install SexLab P+</strong><br>
Download from <a href="https://www.loverslab.com/files/file/25318-sexlab-p/">LoversLab</a> or grab a WIP version from the <a href="https://discord.gg/JPSHb4ebqj">Discord</a>.<br>
<strong>Important:</strong> Let SLP+ overwrite every mod touching base SexLab. It must win all conflicts.
</li>

<li>
<strong>Install Animation Packs</strong><br>
Install SLAL packs (not the loader itself), then overwrite with the <code>Automated.SLSB.Conversions</code> package. See <a href="../animation-packs/">Animation Packs</a> for the full list.
</li>

<li>
<strong>Generate Behaviors</strong><br>
Run your behavior generation tool (Nemesis or Pandora). Note: Pandora has known issues with Draugrs and Skeletons.
</li>

<li>
<strong>Verify Load Order</strong><br>
Double-check that no mods are overwriting SLP+ files unless specifically patched for it.
</li>

<li>
<strong>Test on a New Game</strong><br>
Always test on a fresh save to ensure everything works correctly.
</li>
</ol>
